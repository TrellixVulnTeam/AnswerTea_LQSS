Photograph Web Application
==========================

Description
===========
This is a web application for sorting and browsing photographs and their transformation version!

Prerequisites
=============
*Python3.7
*flask
*ImageMagick
*Python Wand ImageMagick bindings
*MySQL server
*MySQL Python connector


Steps
=====
Perform these steps to run the codes:

1)Start the MySQL server

2)Create the database schema

   a) Start mysqlworkbench
   b) Connect to the database server
   d) Run the SQL queries in the file A1.sql

3)Run the example

   run.py
